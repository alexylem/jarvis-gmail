## Description
Tells you number of unread emails you have on your gmail account, and who sent
the last email.

## Usage
```
You: Vérifie mes emails
Jarvis: Vous avez 2 emails non lus, le dernier est de Alex

You: Vérifie mes emails
Jarvis: Vous n'avez aucun email non lu
```
